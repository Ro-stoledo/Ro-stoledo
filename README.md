# Rodrigo Toledo

Sou formado em Engenharia Civil há 18 anos, com experiência nas áreas de planejamento, custos, construção e operações, atualmente em transição para a área de Análise de Dados.

Ao longo da minha carreira, experienciei diversas formas de tomadas de decisão, incluindo orçamento, controle de custos, acompanhamento de indicadores e comunicação com diferentes stakeholders. Um ponto em comum de todas estas experiências foi a necessidade de sempre trazer resultados e respostas aos mais variados questionamentos para clientes internos ou externos, e foi justamente esta característica que me atraiu para a área de Dados.

Pegar uma massa de dados e tratá-la usando diversas ferramentas, técnicas e criatividade, ajustando e polindo para extrair o máximo possível e então obter o maior número de dados e insights possíveis para que a melhor decisão possa ser tomada é uma atividade que se alinha muito com minha visão.

Atualmente, venho desenvolvendo projetos utilizando ferramentas como Python (Pandas, Matplotlib/Seaborn), SQL e Excel, com foco em análise exploratória de dados, visualização e geração de insights.

Neste repositório estão projetos que simulam problemas reais de negócio, onde aplico essas ferramentas para gerar análises e recomendações baseadas em dados.

Estou em busca de uma oportunidade como Analista de Dados, onde eu possa combinar minha experiência de negócio com habilidades técnicas em dados para gerar impacto real.
